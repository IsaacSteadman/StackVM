DIV (Divide Instruction)

The DIV instructions, number 72 (0x48) through 79 (0x4F), all pop 2, N byte operands off the stack.

Stack Diagram

  • A
  • B
->DIV->
  • Result

The first operand popped off the stack is B, the second popped is A.

All variants push the an N byte unsigned integer onto the stack that represents the equivalent C expression: A / B

DIV1(S) works on a 8 bit integer (N = 1 byte).

DIV2(S) works on a 16 bit integer (N = 2 bytes).

DIV4(S) works on a 32 bit integer (N = 4 bytes).

DIV8(S) works on a 64 bit integer (N = 8 bytes).

DIV1 is assigned opcode 72 (0x48).

DIV1S is assigned opcode 73 (0x49).

DIV2 is assigned opcode 74 (0x4A).

DIV2S is assigned opcode 75 (0x4B).

DIV4 is assigned opcode 76 (0x4C).

DIV4S is assigned opcode 77 (0x4D).

DIV8 is assigned opcode 78 (0x4E).

DIV8S is assigned opcode 79 (0x4F).

AND ("Bitwise And" Instruction)

The AND instructions, number 32 (0x20) through 35 (0x23), all pop 2, N byte operands off the stack.

Stack Diagram

  • A
  • B
->AND->
  • Result

The first operand popped off the stack is B, the second popped is A.

All variants push a N byte unsigned integer onto the stack that represents the equivalent C expression: A & B

AND1 works on an unsigned 8 bit integer (N = 1 byte).

AND2 works on an unsigned 16 bit integer (N = 2 bytes).

AND4 works on an unsigned 32 bit integer (N = 4 bytes).

AND8 works on an unsigned 64 bit integer (N = 8 bytes).

AND1 is assigned opcode 32 (0x20).

AND2 is assigned opcode 33 (0x21).

AND4 is assigned opcode 34 (0x22).

AND8 is assigned opcode 35 (0x23).

LSHIFT ("Left Bitwise Shift" Instruction)

The LSHIFT instructions, number 16 (0x10) through 19 (0x13), all pop a byte off the stack indicating how many bits to shift by.

Stack Diagram

  • A (n bytes)
  • B (1 byte)
->LSHIFT->
  • Result (n bytes)

All variants push the same number of bytes as were popped off minus the byte for the shift. The bits are shift towards the most significant bit.

LSHIFT1 shifts an unsigned 8 bit integer (1 byte).

LSHIFT2 shifts an unsigned 16 bit integer (2 byte).

LSHIFT4 shifts an unsigned 32 bit integer (4 byte).

LSHIFT8 shifts an unsigned 64 bit integer (8 byte).

LSHIFT1 is assigned opcode 16 (0x10).

LSHIFT2 is assigned opcode 17 (0x11).

LSHIFT4 is assigned opcode 18 (0x12).

LSHIFT8 is assigned opcode 19 (0x13).

FSUB ("Floating Point Subtract" Instruction)

The FSUB instructions, number 100 (0x64) through 103 (0x67), all pop 2, N byte operands off the stack.

Stack Diagram

  • A
  • B
->FSUB->
  • Result

The first operand popped off the stack is B, the second popped is A.

All variants push a N byte float onto the stack that represents the equivalent C expression: A - B

FSUB2 works on a 16 bit float (N = 2 bytes).

FSUB4 works on a 32 bit float (N = 4 bytes).

FSUB8 works on a 64 bit float (N = 8 bytes).

FSUB16 works on a 128 bit float (N = 16 bytes).

FSUB2 is assigned opcode 100 (0x64).

FSUB4 is assigned opcode 101 (0x65).

FSUB8 is assigned opcode 102 (0x66).

FSUB16 is assigned opcode 103 (0x67).

FCMP ("Floating Point Compare" Instruction)

The FCMP instructions, number 116 (0x74) through 119 (0x77), all pop 2, N byte operands off the stack.

The first operand popped off the stack is B, the second popped is A.

All variants push a 1 byte signed integer onto the stack that represents the following equivalent C expression:
A < B ? -1 : (A == B ? 0 : 1)

In other words, CMP pushes a negative number if A < B, a zero if A == B, or a positive number if A > B

FCMP2 works on a 16 bit float (N = 2 bytes).

FCMP4 works on a 32 bit float (N = 4 bytes).

FCMP8 works on a 64 bit float (N = 8 bytes).

FCMP16 works on a 128 bit float (N = 16 bytes).

FCMP2 is assigned opcode 116 (0x74).

FCMP4 is assigned opcode 117 (0x75).

FCMP8 is assigned opcode 118 (0x76).

FCMP16 is assigned opcode 119 (0x77).

RST_SP ("Stack Decrement" Instruction)

The RST_SP instructions, number 60 (0x3C) through 63 (0x3F), all pop 1, N byte operand off the stack.

Stack Diagram

  • A
->RST_SP->
  • NONE(Stack Changed)

The operand popped off the stack is A, and there is nothing pushed onto the stack.

The meaning of stack decrement is a reduction of the stack (like a pop, sp increases).

RST_SP1 works on an unsigned 8 bit integer (N = 1 byte).

RST_SP2 works on an unsigned 16 bit integer (N = 2 bytes).

RST_SP4 works on an unsigned 32 bit integer (N = 4 bytes).

RST_SP8 works on an unsigned 64 bit integer (N = 8 bytes).

RST_SP1 is assigned opcode 60 (0x3C).

RST_SP2 is assigned opcode 61 (0x3D).

RST_SP4 is assigned opcode 62 (0x3E).

RST_SP8 is assigned opcode 63 (0x3F).

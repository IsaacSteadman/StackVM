LROT ("Left Bitwise Rotate" Instruction)

The LROT instructions, number 24 (0x18) through 27 (0x1B), all pop a byte off the stack indicating how many bits to rotate by.

Stack Diagram

  • A (n bytes)
  • B (1 byte)
->LROT->
  • Result (n bytes)

All variants push the same number of bytes as were popped off, minus the byte for the shift. The bits are rotated towards the most significant bit.

LROT1 rotates an unsigned 8 bit integer (1 byte).

LROT2 rotates an unsigned 16 bit integer (2 byte).

LROT4 rotates an unsigned 32 bit integer (4 byte).

LROT8 rotates an unsigned 64 bit integer (8 byte).

LROT1 is assigned opcode 24 (0x18).

LROT2 is assigned opcode 25 (0x19).

LROT4 is assigned opcode 26 (0x1A).

LROT8 is assigned opcode 27 (0x1B).

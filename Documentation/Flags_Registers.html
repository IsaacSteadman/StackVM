StackVM System Registers

StackVM's system registers are accessed through the LOAD and STOR instructions with variant SYS_REG

HEX_CODE CODE_NAME Global/Local? Who Can Read? Who Can Write? Additional Notes
0x00 SYS_INFO_G Global Any None Holds Global System Infomation (Number of cores, this core's id ...)
0x01 SYS_INFO_L Local Any PL1 Holds Local Processor Infomation (execution/privilege level ...)
0x02 TRAP_TABLE_G Global Any PL1 Holds Global Trap Table
0x03 TRAP_TABLE_L Local Any PL1 Holds Process Trap Table
0x04 INT_TABLE_G Local Any PL1 Holds Global Interrupt Vector Table (for this processor)
0x05 INT_TABLE_L Local Any PL1 Holds Process Interrupt Vector Table
0x06 SYS_MAIN_PTE Local Any PL1 Holds Global Page Table Entry
0x07 SYS_PROC_PTE Local Any PL1 Holds Process Page Table Entry
0x08 SYS_MAIN_SP Local PL1 PL1 Holds Main stack pointer

SYS_INFO_G

bits[0:15] bits[16:31] bits[32:63]
# of cores this code id RESERVED (0)

SYS_INFO_L

bits[0:5] bits[6:8] bits[9:63]
Priority (P0 - P63) Privilege Level(PL0 - PL7) RESERVED (0)

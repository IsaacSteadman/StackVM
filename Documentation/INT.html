INT (Interrupt Instruction)

INT is instruction 15 (0x0F) and instructs the processor to call the address specified in the Interrupt Descriptor Table.

The Interrupt code is specified as an immediate byte following the INT instruction

The Interrupt Descriptor Table (IDT) is a contiguous array of 256 8-byte pointers to executable code.

The first 32 (0x00 thru 0x3F) interrupts are reserved for system* use.

*System: system use may include hardware defined behavior instead of a call to the code address specified in the IDT

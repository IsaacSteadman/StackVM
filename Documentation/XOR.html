XOR ("Bitwise Xor" Instruction)

The XOR instructions, number 44 (0x2C) through 47 (0x2F), all pop 2, N byte operands off the stack.

  • B
->XOR->
  • Result

The first operand popped off the stack is B, the second popped is A.

All variants push a N byte unsigned integer onto the stack that represents the equivalent C expression: A ^ B

XOR1 works on an unsigned 8 bit integer (N = 1 byte).

XOR2 works on an unsigned 16 bit integer (N = 2 bytes).

XOR4 works on an unsigned 32 bit integer (N = 4 bytes).

XOR8 works on an unsigned 64 bit integer (N = 8 bytes).

XOR1 is assigned opcode 44 (0x2C).

XOR2 is assigned opcode 45 (0x2D).

XOR4 is assigned opcode 46 (0x2E).

XOR8 is assigned opcode 47 (0x2F).

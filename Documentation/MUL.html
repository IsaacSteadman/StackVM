MUL (Multiply Instruction)

The MUL instructions, number 64 (0x40) through 71 (0x47), all pop 2, N byte operands off the stack.

Stack Diagram

  • A
  • B
->MUL->
  • Result

The first operand popped off the stack is B, the second popped is A.

All variants push a N byte unsigned integer onto the stack that represents the equivalent C expression: A * B

MUL1(S) works on a 8 bit integer (N = 1 byte).

MUL2(S) works on a 16 bit integer (N = 2 bytes).

MUL4(S) works on a 32 bit integer (N = 4 bytes).

MUL8(S) works on a 64 bit integer (N = 8 bytes).

MUL1 is assigned opcode 64 (0x40).

MUL1S is assigned opcode 65 (0x41).

MUL2 is assigned opcode 66 (0x42).

MUL2S is assigned opcode 67 (0x43).

MUL4 is assigned opcode 68 (0x44).

MUL4S is assigned opcode 69 (0x45).

MUL8 is assigned opcode 70 (0x46).

MUL8S is assigned opcode 71 (0x47).

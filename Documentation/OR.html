OR ("Bitwise Or" Instruction)

The OR instructions, number 36 (0x24) through 39 (0x27), all pop 2, N byte operands off the stack.

  • B
->OR->
  • Result

The first operand popped off the stack is B, the second popped is A.

All variants push a N byte unsigned integer onto the stack that represents the equivalent C expression: A | B

OR1 works on an unsigned 8 bit integer (N = 1 byte).

OR2 works on an unsigned 16 bit integer (N = 2 bytes).

OR4 works on an unsigned 32 bit integer (N = 4 bytes).

OR8 works on an unsigned 64 bit integer (N = 8 bytes).

OR1 is assigned opcode 36 (0x24).

OR2 is assigned opcode 37 (0x25).

OR4 is assigned opcode 38 (0x26).

OR8 is assigned opcode 39 (0x27).

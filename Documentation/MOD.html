MOD (Modulus Instruction)

The MOD instructions, number 80 (0x50) through 87 (0x57), all pop 2, N byte operands off the stack.

Stack Diagram

  • A
  • B
->MOD->
  • Result

The first operand popped off the stack is B, the second popped is A.

All variants push the an N byte unsigned integer onto the stack that represents the equivalent C expression: A % B

MOD1(S) works on a 8 bit integer (N = 1 byte).

MOD2(S) works on a 16 bit integer (N = 2 bytes).

MOD4(S) works on a 32 bit integer (N = 4 bytes).

MOD8(S) works on a 64 bit integer (N = 8 bytes).

MOD1 is assigned opcode 80 (0x50).

MOD1S is assigned opcode 81 (0x51).

MOD2 is assigned opcode 82 (0x52).

MOD2S is assigned opcode 83 (0x53).

MOD4 is assigned opcode 84 (0x54).

MOD4S is assigned opcode 85 (0x55).

MOD8 is assigned opcode 86 (0x56).

MOD8S is assigned opcode 87 (0x57).

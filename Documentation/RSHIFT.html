RSHIFT ("Right Bitwise Shift" Instruction)

The RSHIFT instructions, number 20 (0x14) through 23 (0x17), all pop a byte off the stack indicating how many bits to shift by.

Stack Diagram

  • A (n bytes)
  • B (1 byte)
->RSHIFT->
  • Result (n bytes)

All variants push the same number of bytes as were popped off minus the byte for the shift. The bits are shift towards the least significant bit.

RSHIFT1 shifts an unsigned 8 bit integer (1 byte).

RSHIFT2 shifts an unsigned 16 bit integer (2 byte).

RSHIFT4 shifts an unsigned 32 bit integer (4 byte).

RSHIFT8 shifts an unsigned 64 bit integer (8 byte).

RSHIFT1 is assigned opcode 20 (0x14).

RSHIFT2 is assigned opcode 21 (0x15).

RSHIFT4 is assigned opcode 22 (0x16).

RSHIFT8 is assigned opcode 23 (0x17).

FMUL ("Floating Point Multiply" Instruction)

The FMUL instructions, number 104 (0x68) through 107 (0x6B), all pop 2, N byte operands off the stack.

Stack Diagram

  • A
  • B
->FMUL->
  • Result

The first operand popped off the stack is B, the second popped is A.

All variants push a N byte float onto the stack that represents the equivalent C expression: A * B

FMUL2 works on a 16 bit float (N = 2 bytes).

FMUL4 works on a 32 bit float (N = 4 bytes).

FMUL8 works on a 64 bit float (N = 8 bytes).

FMUL16 works on a 128 bit float (N = 16 bytes).

FMUL2 is assigned opcode 104 (0x68).

FMUL4 is assigned opcode 105 (0x69).

FMUL8 is assigned opcode 106 (0x6A).

FMUL16 is assigned opcode 107 (0x6B).

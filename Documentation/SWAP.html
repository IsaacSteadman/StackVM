SWAP (Swap Instruction)

SWAP is instruction 9 and instructs the processor to swap items on the stack that may be different sizes as specified in the following byte.

Stack Diagram

  • A
  • B
->SWAP->
  • B
  • A

Assume B is on the top of A at the top of the stack.
In other words B is popped off the stack first, then A.

A table representing the bits in the byte is displayed below.

Bit # 7 6 5 4 3 2 1 0
Meaning RESERVED Size B Size A

A table mapping each size to its corresponding size code is displayed below.

Size BitCode
1 byte 0b000 (0x0)
2 bytes 0b001 (0x1)
4 bytes 0b010 (0x2)
8 bytes 0b011 (0x3)
16 bytes 0b100 (0x4)
32 bytes 0b101 (0x5)
64 bytes 0b110 (0x6)
128 bytes 0b111 (0x7)

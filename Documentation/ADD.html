ADD (Add Instruction)

The ADD instructions, number 48 (0x30) through 51 (0x33), all pop 2, N byte operands off the stack.

  • A
  • B
->ADD->
  • Result

The first operand popped off the stack is B, the second popped is A.

All variants push a N byte unsigned integer onto the stack that represents the equivalent C expression: A + B

ADD1 works on an unsigned 8 bit integer (N = 1 byte).

ADD2 works on an unsigned 16 bit integer (N = 2 bytes).

ADD4 works on an unsigned 32 bit integer (N = 4 bytes).

ADD8 works on an unsigned 64 bit integer (N = 8 bytes).

ADD1 is assigned opcode 48 (0x30).

ADD2 is assigned opcode 49 (0x31).

ADD4 is assigned opcode 50 (0x32).

ADD8 is assigned opcode 51 (0x33).

FMOD ("Floating Point Modulus" Instruction)

The FMOD instructions, number 112 (0x70) through 115 (0x73), all pop 2, N byte operands off the stack.

Stack Diagram

  • A
  • B
->FMOD->
  • Result

The first operand popped off the stack is B, the second popped is A.

All variants push the an N byte float onto the stack that represents the equivalent C expression: A % B

FMOD2 works on a 16 bit float (N = 2 bytes).

FMOD4 works on a 32 bit float (N = 4 bytes).

FMOD8 works on a 64 bit float (N = 8 bytes).

FMOD16 works on a 128 bit float (N = 16 bytes).

FMOD2 is assigned opcode 112 (0x70).

FMOD4 is assigned opcode 113 (0x71).

FMOD8 is assigned opcode 114 (0x72).

FMOD16 is assigned opcode 115 (0x73).

SUB (Subtract Instruction)

The SUB instructions, number 52 (0x34) through 55 (0x37), all pop 2, N byte operands off the stack.

  • B
->SUB->
  • Result

The first operand popped off the stack is B, the second popped is A.

All variants push a N byte unsigned integer onto the stack that represents the equivalent C expression: A - B

SUB1 works on an unsigned 8 bit integer (N = 1 byte).

SUB2 works on an unsigned 16 bit integer (N = 2 bytes).

SUB4 works on an unsigned 32 bit integer (N = 4 bytes).

SUB8 works on an unsigned 64 bit integer (N = 8 bytes).

SUB1 is assigned opcode 52 (0x34).

SUB2 is assigned opcode 53 (0x35).

SUB4 is assigned opcode 54 (0x36).

SUB8 is assigned opcode 55 (0x37).

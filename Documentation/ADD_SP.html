ADD_SP ("Stack Increment" Instruction)

The ADD_SP instructions, number 56 (0x38) through 59 (0x3B), all pop 1, N byte operand off the stack.

Stack Diagram

  • A
->ADD_SP->
  • NONE(Stack Changed)

The operand popped off the stack is A, and there is nothing pushed onto the stack.

The meaning of stack increment is an addition of the stack (like a push, sp decreases).

ADD_SP1 works on an unsigned 8 bit integer (N = 1 byte).

ADD_SP2 works on an unsigned 16 bit integer (N = 2 bytes).

ADD_SP4 works on an unsigned 32 bit integer (N = 4 bytes).

ADD_SP8 works on an unsigned 64 bit integer (N = 8 bytes).

ADD_SP1 is assigned opcode 56 (0x38).

ADD_SP2 is assigned opcode 57 (0x39).

ADD_SP4 is assigned opcode 58 (0x3A).

ADD_SP8 is assigned opcode 59 (0x3B).

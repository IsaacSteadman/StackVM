JMP (Jump Instruction)

The JMP instructions, number 56 (0x38) through 59 (0x3B), all pop 1, N byte operand off the stack.

->JMP->

The first operand popped off the stack is 8 byte integer A, and there is nothing pushed onto the stack.

If the instruction is conditional a 1 byte boolean B is popped off the stack, otherwise A is the only operand popped off the stack.

If the instruction is relative, A is signed, otherwise A is unsigned.

JMP performs an absolute unconditional jump (ip = A).

JMPIF performs an absolute conditional jump (ip = B ? A : ip).

RJMP performs a relative unconditional jump (ip += A).

RJMPIF performs a relative conditional jump (ip += B ? A : 0).

JMP is assigned opcode 120 (0x78).

JMPIF is assigned opcode 121 (0x79).

RJMP is assigned opcode 122 (0x7A).

RJMPIF is assigned opcode 123 (0x7B).

NOT ("Bitwise Not" Instruction)

The NOT instructions, number 40 (0x28) through 43 (0x2B), all pop 1, N byte operand off the stack.

Stack Diagram

  • Input
->NOT->
  • Output

The operand popped off the stack is A.

All variants push the an N byte unsigned integer onto the stack that represents the equivalent C expression: ~A

NOT1 works on an unsigned 8 bit integer (N = 1 byte).

NOT2 works on an unsigned 16 bit integer (N = 2 bytes).

NOT4 works on an unsigned 32 bit integer (N = 4 bytes).

NOT8 works on an unsigned 64 bit integer (N = 8 bytes).

NOT1 is assigned opcode 40 (0x28).

NOT2 is assigned opcode 41 (0x29).

NOT4 is assigned opcode 42 (0x2A).

NOT8 is assigned opcode 43 (0x2B).

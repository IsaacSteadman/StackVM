FADD ("Floating Point Add" Instruction)

The FADD instructions, number 96 (0x60)) through 99 (0x63), all pop 2, N byte operands off the stack.

Stack Diagram

  • A
  • B
->FADD->
  • Result

The first operand popped off the stack is B, the second popped is A.

All variants push a N byte float onto the stack that represents the equivalent C expression: A + B

FADD2 works on a 16 bit float (N = 2 bytes).

FADD4 works on a 32 bit float (N = 4 bytes).

FADD8 works on a 64 bit float (N = 8 bytes).

FADD16 works on a 128 bit float (N = 16 bytes).

FADD2 is assigned opcode 96 (0x60).

FADD4 is assigned opcode 97 (0x61).

FADD8 is assigned opcode 98 (0x62).

FADD16 is assigned opcode 99 (0x63).
